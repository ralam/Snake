# Snake

[Link to live site][site]

[site]: http://ralam.github.io/Snake/

The classic game of Snake, built using jQuery and CSS. High score is stored locally for each session.

Controls:
 - Start a new game: `spacebar`
 - Change direction: `← ↑ → ↓`
 - Pause/unpause: `ESC`
